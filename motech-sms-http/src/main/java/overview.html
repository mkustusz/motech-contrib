Template definition (sms-http-template.json) 
request: specifies the structure of the http request exposed by an sms provider for sendinf sms
    urlPath: the url minus the query string
    queryParameters: a hash of query parameters to add to the request
response: specifies the structure of the response body returned by the request
    success: the response body returned by the request on successful delivery of the sms
$recipients: placeholder for recipients of the message; filled in from sendSMS() argument
$message: placeholder for message text; filled in from sendSMS() argument
